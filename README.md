# Affective Positioning System (APS)

## Overview
The Affective Positioning System (APS) is a conceptual and computational framework designed to analyze relationship dynamics across three orthogonal dimensions:

1. Internal State (affective stability & perceived need)
2. Power Dynamics (dependency asymmetry)
3. Motivational Structure (resource vs emotional orientation)

This system transforms abstract relational experiences into structured, interpretable models.


## Key Idea

Relationships are not only emotional — they are structured systems shaped by:

- Dependency
- Replaceability (alternatives)
- Power asymmetry
- Motivational alignment

APS allows users to **locate themselves within these structures**, rather than relying on narrative intuition.


## Features

- Multi-dimensional relationship analysis
- Dependency structure modeling
- Power-risk estimation
- Narrative–structure mismatch detection
- Interactive interface (Streamlit-based)


## Example Output

- Dominant dependency type
- Relationship structure classification
- Risk level (e.g., asymmetric dependency)
- Structural vs emotional mismatch
